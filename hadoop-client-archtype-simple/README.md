Hadoop Client Simple Archtype
=============================

USAGE
-----

Install archtype:

	`cd ./hadoop-client-archtype-simple`
	`mvn install`


Ref. [Archetype creation](http://maven.apache.org/archetype/maven-archetype-plugin/advanced-usage.html)
